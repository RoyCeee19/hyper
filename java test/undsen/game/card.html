ГРАФИК КАРТ

Компьютерийг монголчууд тоглоход их ашиглах болсон. Харин тоглохын тулд график карт зайлшгүй хэрэгтэй байх ажээ. Гэхдээ:

  • 1. График процессорын ерөнхий хурд
  • 2. Санах ойн мэдээлэл дамжуулах хурд
  • 3. Мэдээлэл дамжуулах багтаамж зэргийг мэдэх ёстой.

Мөн график картанд гол хамаарах зүйл бол дэлгэцийн хэмжээ том байх ёстой ба хэрэв 15″ дэлгэцтэй бол өндөр чанарын график карт худалдаж авах шаардлагагүй.
