RAM буюу САНАХ ОЙ

RAM буюу санах ой нь 256 мб-аас дээш байх ёстойг та андахгүй мэддэг болсон. Харин дараах зүйлийг анхаарах хэрэгтэй. Үүнд:

  • 1. Memory capacity буюу шуурхай санах ойн мэдээлэл агуулах ерөнхий багтаамж
  • 2. Memory speed буюу шуурхай санах ойн мэдээлэл дамжуулах хурд

Санах ойг эх хавтангийн дэмжих боломжит хурдны дээд хэмжээгээр тохируулж тавих хэрэгтэй. Өөрөөр хэлбэл DDR266, DDR333, DDR400 хурд бүхий санах ой дэмждэг эх хавтанд PC3200 буюу DDR400 санах ой тавибал таны компьютерийн хурд үнэн “тасархай” болох ажээ.
